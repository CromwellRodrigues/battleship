# 🚢 Battleship Game 🎯

Welcome to the **Battleship Game** project! 

## 🕹️ How to Play
- Place your fleet on the board 🚢
- Try to sink the enemy ships before they sink yours! 💥
- Click on the enemy grid to attack 🎯
- Use the **Rotate Ship** button to change ship orientation 🔄

## 🧩 Features
- Responsive design 📱💻
- Colorful grids and interactive UI 🎨
- Fun sound effects (optional) 🔊
- Play against the computer 🤖

## 🚀 Getting Started
1. Clone the repo
2. Run `npm install` (if needed)
3. Open `index.html` in your browser 🌐

## 🛠️ Tech Stack
- JavaScript
- Tailwind CSS
- HTML5
